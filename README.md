## 🚀 Como utilizar este material

Este material foi construído para ser visualizado via **MkDocs**.
Para rodar localmente:

1. Clone o repositório.
2. Instale as dependências: `pip install -r requirements.txt` (ou use o `pyproject.toml`).
3. Execute: `mkdocs serve`.
